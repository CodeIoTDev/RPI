MQTT

Open Terminal and enter the following commands
(GLOBALLY)
sudo apt-get update
sudo apt-get install python3-rpi.gpio
sudo apt install mosquitto mosquitto-clients

(LOCALLY MAIN TERMINAL (T-1))
cd Desktop
cd iot_6
mkdir .venv
cd .venv
python3 -m venv .venv
ls -l
source .venv/bin/activate
pip install paho-mqtt 

• In a new terminal, use following commands (T - 2) (OPEN GLOBALLY)
mosquitto_sub -h test.mosquitto.org -p 1883 -t test/led_control
• This is the terminal for subscribing, we will call this the MQTT subscriber

• In a new terminal, use following commands (T - 3) (OPEN GLOBALLY)
mosquitto_pub -h test.mosquitto.org -p 1883 -t test/led_control -m "Hello, IoT World"
• This is the terminal for publishing, we will call this the MQTT publisher

• The message will be displayed on the subscriber terminal
• Run the python code on the Main terminal python3 code.py

• Send “ON” from publishing terminal
• Message is displayed on subscribing terminal
• Message is also displayed on main terminal
• LED is turned on.

• Send “OFF” from publishing terminal
• Message is displayed on subscribing terminal
• Message is also displayed on main terminal
• LED is turned off.
